Gráficos com barras

Gráficos com barras lado a lado na vertical

Para criar esse gráfico você precisa criar um canva e definir um id para ele 

<canvas id="chartBar" width="500px"></canvas>

O próximo passo é necessário escrever o código responsável por criar o gráfico. 


let ctx = `chartBar`;              
new Chart(ctx, {
  type: 'bar', // Esse é o tipo do gráfico
  data: {
    labels: ['2022-05-10', '2022-05-11', '2022-05-12'], // As legendas referentes aos valores no gráfico
    datasets: [
      // Os valores dos gráficos
      {
        label: 'Parte 1',
        data: [1, 2, 3],
        backgroundColor: '#4287f5',
      },
      {
        label: 'Parte 2',
        data: [4, 5, 6],
        backgroundColor: '#2ebf52',
      },
      {
        label: 'Parte 3',
        data: [7, 8, 9],
        backgroundColor: '#bf2e2e',
      },
    ],
  },
  options: {
    responsive: true,
    indexAxis: 'x', // Responsável por deixar o gráfico com as barras na vertical, as informações seguem pelo eixo "x"
  },
});

Pronto, gráficos estão renderizados conforme mostrado abaixo.

Gráficos com barras agrupadas na vertical

Para criar esse gráfico você precisa criar um canva e definir um id para ele 

<canvas id="chartBarGroup" width="500px"></canvas>

O próximo passo é necessário criar a função que vai criar o gráfico. 


let ctx = `chartBarGroup`;              
new Chart(ctx, {
  type: 'bar', // Esse é o tipo do gráfico
  data: {
    labels: ['2022-05-10', '2022-05-11', '2022-05-12'], // As legendas referentes aos valores no gráfico
    datasets: [
      // Os valores dos gráficos
      {
        label: 'Parte 1',
        data: [1, 2, 3],
        backgroundColor: '#4287f5',
      },
      {
        label: 'Parte 2',
        data: [4, 5, 6],
        backgroundColor: '#2ebf52',
      },
      {
        label: 'Parte 3',
        data: [7, 8, 9],
        backgroundColor: '#bf2e2e',
      },
    ],
  },
  options: {
    responsive: true,
    indexAxis: 'x', // Responsável por deixar o gráfico com as barras na vertical, as informações seguem pelo eixo "x"
  },
  //Adicionado essa parte abaixo no código, as barras ficam agrupadas
  scales: {
            y: {
              stacked: true,
            },
            x: {
              stacked: true,
            },
  },
});
